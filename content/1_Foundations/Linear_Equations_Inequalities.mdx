## Overview

Linear equations ($ax + b = c$) and linear inequalities ($ax + b < c$) form the
backbone of AMC 8 algebra. This module covers solving techniques, handling
fractions and decimals in equations, and the key rule that flipping an inequality
sign is required when multiplying or dividing by a negative number.

## Key Ideas

- Isolate the variable: combine like terms, then undo addition/subtraction, then
  undo multiplication/division.
- When you multiply or divide both sides of an inequality by a negative number,
  flip the inequality sign.
- Check solutions by substituting back into the original equation.
- Compound inequalities like $2 < x + 1 < 7$ can be solved by subtracting 1
  from all three parts.

## Core Skills

### One-Step and Two-Step Equations

Undo operations in reverse order of operations.

### Equations with Fractions

Clear denominators by multiplying both sides by the LCD.

### Inequality Graphing

Represent solution sets on a number line using open/closed circles.

## Worked Example

Solve $3x - 7 \le 2x + 5$.

Subtract $2x$: $x - 7 \le 5$. Add 7: $x \le 12$.
