## Overview

Many AMC 8 mistakes come from sign errors with negative numbers or
misunderstanding absolute value. This module builds reliable habits for integer
arithmetic and introduces absolute value as distance on the number line.

## Key Ideas

- $|x|$ equals $x$ if $x \ge 0$ and $-x$ if $x < 0$. It measures distance from
  zero.
- $|a - b|$ is the distance between $a$ and $b$ on the number line.
- Multiplying two negatives gives a positive; multiplying a positive and a
  negative gives a negative.
- Subtracting a negative is the same as adding a positive:
  $a - (-b) = a + b$.

## Core Skills

### Integer Arithmetic Rules

Keep track of signs through chains of operations using the sign rules for
multiplication and division.

### Absolute Value Equations

$|x| = 5$ means $x = 5$ or $x = -5$. Split into cases.

### Distance on the Number Line

The distance from $-3$ to $4$ is $|{-3} - 4| = 7$.

## Worked Example

Evaluate $|{-7}| - |3 - 10|$.

$|{-7}| = 7$ and $|3 - 10| = |{-7}| = 7$. So the answer is $7 - 7 = 0$.
